# Dinesh Karthik Mulumudi Personal Website

This repository hosts the source files for my personal academic website. It includes:

- A clean, minimalist HTML page (`index.html`).
- Simple CSS styling (`style.css`).
- Academic project reports and posters located in the `pdfs/` folder.
- Contact information and links to my CV, GitHub, and email.

## How to view locally

```bash
# Serve the site locally (requires Python 3)
python3 -m http.server 8000 --directory .
```

Open a browser and navigate to `http://localhost:8000`.

## Deployment

The site is automatically served via GitHub Pages from the `main` branch of this repository.
